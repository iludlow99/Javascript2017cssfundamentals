Id's

An Id is a unique identifier

Id notes

  • Is unique to the element that it's on
  • To target it from our css we need to use a # sign
  • You can use them with classes on the same element
  • Uses: headers or about sections
  • Generally, classes are used for html, while Id's are used for css
  • Id's outweigh classes
  • They should only be used once per page
